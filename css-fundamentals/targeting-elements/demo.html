Relative Units

This is 50px large

em is relative to the font-size of parent if we're using it for font properties, but it's relative to the font-size of the element itself if being used for anything not font related This element's font-size is 0.5em x 50px = 25px large The border is 0.5em x the element's font size (25px) = 12.5px

If the root element's font-size is 16px, then 1.5rem is 24px Measuring everything off of the same unit, such as when using rem's might be easier in some cases
